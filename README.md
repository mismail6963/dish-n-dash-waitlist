# Dish N Dash — Founders Club Waitlist

**Official waitlist website for the Dish N Dash Founders Club.**

Dish N Dash is a Saudi-born, AI-powered dining platform that combines real-time booking, social discovery, pre-ordering, and built-in loyalty — so diners skip the chaos and restaurants keep every seat full.

This repository contains the interactive waitlist website designed to build anticipation, create exclusivity, and convert early adopters into Founding Members before the full app launch.

---

## About Dish N Dash

Dish N Dash is a unified dining platform for Saudi Arabia's mid-to-high-end restaurant market. It replaces fragmented tools with a single experience:

- **Reservations** — Real-time availability with instant confirmation
- **Pre-orders & Payments** — Reduce wait times and drop-offs
- **Order Automation** — Requests sent directly to POS systems, minimizing errors
- **Customer Insights & Loyalty** — AI-driven analytics for targeted offers and retention

The platform launches first in Riyadh, targeting urban diners aged 15–35 and fine dining / high-end restaurants.

## Waitlist Website Overview

The website is a premium, interactive experience that guides users toward joining the **Founders Club** — a limited early-access program for the first wave of Dish N Dash users.

### Preloader

A short animation introduces the brand concept:

1. The Dish N Dash logo forms from small glowing data points
2. Dots rearrange into restaurant tables and seats
3. Tables reorganize themselves intelligently
4. Shapes merge back into the logo

Text fades in: *"Dining, intelligently organized."*

### Bubble Menu (Main Navigation)

The homepage centers on an animated bubble cluster with three orbiting bubbles:

- **What is Dish N Dash**
- **How It Works**
- **Join the Waitlist**

The Waitlist bubble is always the largest and subtly grows on any hover interaction, creating a natural pull toward signup.

### What is Dish N Dash

An animated storytelling section using scroll-driven effects:

- **Scroll Float** — UI elements (restaurant cards, reservation times, dish images) move at different parallax speeds
- **True Focus** — Only the current message stays sharp; surrounding content blurs
- **Scroll Reveal** — Content appears progressively: Discover → Reserve → Pre-order → Dine
- **Rotating Text** — Headline cycles through: *Discover, Reserve, Pre-Order, Dine, Repeat*

### How It Works (Exclusive Access Mechanic)

An intentionally "locked" page that drives curiosity:

1. Users see a *"404 — Page Locked"* message with fuzzy text animation
2. Below: *"Some dining experiences shouldn't be available to everyone."*
3. An **Upgrade** button opens a modal explaining early access
4. Clicking **Proceed** redirects to the waitlist

### VIP Founders Club Signup

The waitlist is framed as an exclusive program:

> **"Join the Dish N Dash Founders Club"**
> *A limited group of early diners shaping the future of dining in Riyadh.*

Perks include:
- Early access to the app
- Exclusive restaurant drops
- Priority reservations at partner restaurants
- Founding member badge inside the app

### Post-Signup Referral System

After joining, users see their waitlist position and can move up by inviting friends:

- Shareable referral link
- WhatsApp and Instagram share buttons
- Live position tracker
